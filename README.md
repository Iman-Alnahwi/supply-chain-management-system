# Supply Chain Management System

A standalone desktop application designed to streamline supply chain processes, manage supplier databases, and handle product orders efficiently. Built with a user-friendly graphical interface and secure data persistence.

## Features
* Secure Authentication:** Professional login system for system administrators and managers.
* Supplier Management:** Full CRUD operations to track, add, and manage supplier records.
* Product & Order Tracking:Efficient handling of supply items, product details, and order history.
* Database Persistence:** Fully integrated with a relational database backend.

## Tech Stack
* Language: Java
* GUI Framework: Java Swing (NetBeans GUI Builder)
* Database: MySQL & MySQL Workbench
* Environment: JDK 21

## Database Setup
To run this project locally, import the provided `database_setup.sql` script into your MySQL Workbench instance before running the application.
